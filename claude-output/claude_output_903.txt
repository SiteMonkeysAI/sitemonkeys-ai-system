---SUMMARY---

**Root Cause:** The PostgreSQL `pg` driver returns `DECIMAL/NUMERIC` column values as JavaScript strings by default. The `relevance_score` column (defined as `DECIMAL(3,2)`) was being returned from the database as a string like `"0.95"` rather than the number `0.95`.

In `calculateHybridScore()` (line 771 of `api/services/semantic-retrieval.js`), the condition `memory.relevance_score >= 0.90` silently passed due to JavaScript's implicit type coercion of strings to numbers for comparison operators. However, `memory.relevance_score.toFixed(2)` then crashed because `String.prototype.toFixed` does not exist — only `Number.prototype.toFixed` does.

This exception propagated out of `calculateHybridScore`, through the `.map()` call at line 2005–2008, and was caught by the outer `try/catch` at line 2526, causing the entire retrieval to return `success: false` with `memories: []` on every query — hence zero memories injected on all calls.

**Fix Applied:** In `api/services/semantic-retrieval.js`, within the `else` branch of `calculateHybridScore()` (the non-high-priority path that applies the MEM-007 importance boost), the raw `memory.relevance_score` is now converted to a proper JavaScript number via `parseFloat()` before any numeric operations:

```js
// Before (broken):
if (memory.relevance_score && memory.relevance_score >= 0.90) {
  const importanceBoost = (memory.relevance_score - 0.90) * 6.0;
  score += importanceBoost;
  console.log(`...relevance_score=${memory.relevance_score.toFixed(2)}...`);  // CRASH
}

// After (fixed):
const storedRelevanceScore = parseFloat(memory.relevance_score) || 0;
if (storedRelevanceScore >= 0.90) {
  const importanceBoost = (storedRelevanceScore - 0.90) * 6.0;
  score += importanceBoost;
  console.log(`...relevance_score=${storedRelevanceScore.toFixed(2)}...`);  // OK
}
```

This is a one-file, minimal change that fixes the type mismatch at the first point of use. The scoring logic (threshold, boost formula, cap values) is unchanged — only the type normalization is added.

---END SUMMARY---
